Description

Persona is a one page responsive HTML5 vCard template. It has clean view, simple structure and it's focused on YOU! Template build with HTML5 simplicity, CSS3 effects and jQuery power. It's responsive, so it looks good on mobile devices.

Persona can be used not only for your personal site it's also perfect choise for your small creative agency.

All the html, css, js code are well organized and commented that allows you to make changes without any problems.

HTML Structure

Persona has responsive layout. Here is the general structure.

<div class="wrapper vcard">

    <!-- BEGIN NAV -->
    <nav class="nav">
        <div class="container">
            <div class="nav-trigger"></div>
            <ul>
                <li class="current"><a href="#about">About me</a></li>
                <li><a href="#resume">Resume</a></li>
                <li><a href="#portfolio">Portfolio</a></li>
                <li><a href="#testimonials">Testimonials</a></li>
                <li><a href="#contacts">Contacts</a></li>
            </ul>
        </div>
    </nav>
    <!-- END NAV -->

    <!-- BEGIN CONTENT -->
    <div class="main">
        <!-- Content -->
    </div>
    <!-- END CONTENT -->

</div>

Content block consists of different sections. By default there are 5 sections.

  • About
  • Resume
  • Portfolio
  • Testimonials
  • Contacts

Custom sections

If you would like to add your own sections, simply add the following code:

<section id="Section-ID" class="cont-sec">

    <div class="sec-title">
        <div class="container">
            <h2>Section title</h2>
        </div>
    </div>

    <div class="container">
        <!-- Section content -->
    </div>

</section>

Then add new menu item with same hash as the section id.

<li><a href="#Section-ID">Section title</a></li>

CSS Files

Template uses two main CSS files.

<link rel="stylesheet" href="assets/css/normalize.css">
<link rel="stylesheet" href="assets/css/main.css">

The first one is a normalize.css. Normalize.css makes browsers display elements more consistently and in line with modern standards. It's good alternative to old CSS resets. Second file contains main stylings for the page.

Also there is one additional css file for custom content scroller script.

<link rel="stylesheet" href="assets/plugins/mCustomScrollbar/jquery.mCustomScrollbar.css">

Media queries

Template uses some media queries for different size screens. The built in media queries include:

  • Tablet Portrait. Between 400px and 980px
  • Mobile Portrait. Less then 400px

External javascript plugins

Template includes nine external third party libraries.

Modernizr

<script src="assets/js/modernizr-2.6.2.min.js"></script>

Modernizr is a JavaScript library that detects HTML5 and CSS3 features in the user’s browser. Also Modernizr runs through a little loop in JavaScript to enable the various elements from HTML5 for styling in Internet Explorer.

CSS3 PIE

.photo, .photo img, .btn, .btn-2, .filter li, .skill, .skill .progress, .date{behavior:url("assets/js/PIE.htc")}

CSS3 PIE makes Internet Explorer 6-9 capable of rendering several of the most useful CSS3 decoration features.

jQuery One Page Navigation Plugin

<script src="assets/js/jquery-ui-1.9.2.custom.min.js"></script>
<script src="assets/plugins/ope-page-nav/jquery.scrollTo.js"></script>
<script src="assets/plugins/ope-page-nav/jquery.nav.js"></script>

A lightweight jQuery plugin for the navigation on one-page sites.

To change scroll speed, find Main navigation scroll section in script.js file and change the scrollSpeed param.

// Main navigation scroll
$('nav.nav').onePageNav({
    scrollSpeed:500,
    scrollOffset:59,
    easing:'easeInOutQuart',
    begin: function() {
        //Hack so you can click other menu items after the initial click
        $('body').append('
');
    },
    end: function() {
        $('#device-dummy').remove();
    }
});

jQuery custom content scroller

<script src="assets/plugins/mCustomScrollbar/jquery.mCustomScrollbar.js"></script>

Custom scrollbar plugin.

To make custom scrollbar simply add .scroll css class to the element you want. Any element you’re attaching the custom scrollbar should have a typical styling of an overflowed block, e.g. a height (or max-height) value, overflow: auto (or hidden) and its content should be long enough to require a scrollbar.

If you would like to edit «Education» and «Employment» blocks height in «Resume» section, find and change the following style in the style sheet:

.exp{height:380px}

Google Maps API

<script src="http://maps.google.com/maps/api/js?sensor=false"></script>

To change map location, you need to write your own latitude and longitude in scripts.js.

// Latitude and longitude for your location goes here
lat = 55.750497,
lng = 37.616243,

To know your coordinates go to the Google maps page, right-click on the place you want on the map, select "What's here?" and the coordinates will appear in search box.

You can read more about Google Map API here.

Display latest tweets on site.

To display your last tweets, you need to create twitter timeline widget here. Just create a widget with default settings. Then copy the widget id from the resulting html code to the data-widget-id param in the template tweets section.

<!-- BEGIN TWEETS -->
<div class="tweet">

    <a class="twitter-timeline"  href="https://twitter.com/YOUR-TWITTER-USERNAME" data-widget-id="YOUR-TWITTER-TIMELINE-WIDGET-ID" data-theme="dark" data-link-color="#72B5E5" data-chrome="noheader nofooter noborders transparent" data-tweet-limit="3">Tweets by @YOUR-TWITTER-USERNAME</a>

</div>
<!-- END TWEETS -->

Also replace «YOUR-TWITTER-USERNAME» with your username.

PHP files

Еemplate uses a single php file that is needed for the contact form. To make contact form work properly, you need to change adress of the recipient in the send.php file.

$to = "mail@yoursite.com"; // Change the recipient email address to your address.
